# MediaBox

![Image of MediaBox](MediaBox.jpg)


# About
  MediaBox is a Docker-Compose file consiting of all the relevant componenets to setup your own Media Server.

## Includes
* Plex
* Sonarr
* Radarr
* Deluge
* Ombi
* Tautulli
* Cardigann
* TV Headend
* Channels DVR

## Requirements
* Docker
* Docker-Compose
